## 2. Data Structures

### Linear Data Structures
*   **[Array](array.md)**: Contiguous elements accessed by index.
*   **[Linked List](linked-list.md)**: Chain of nodes connected by pointers.
*   **[Array vs. Linked List](array-vs-linked-list.md)**: Core trade-offs between linear structures.
*   **[Stack](stack.md)**: Last-In, First-Out (LIFO) list.
*   **[Queue](queue.md)**: First-In, First-Out (FIFO) list.
*   **[Hash Table](hash-table.md)**: Fast key-value mapping using hash functions.

### Non-Linear & Hierarchical Data Structures
*   **[Binary Tree](binary-tree.md)**: Tree where nodes have up to two children.
*   **[Binary Search Tree (BST)](binary-search-tree.md)**: Sorted binary tree for quick search/insert.
*   **[Heap & Priority Queue](heap-and-priority-queue.md)**: Tree structure for tracking extreme values.
*   **[Trie (Prefix Tree)](trie.md)**: Efficient search tree for string keys.
*   **[Graph](graph.md)**: Network of nodes connected by edges.
*   **[Union-Find (DSU)](union-find.md)**: Tracks and merges disjoint subsets.

## 3. Algorithms & Search

### Search Algorithms
*   **[Binary Search](binary-search.md)**: Efficient O(log n) search on sorted arrays.
*   **[Breadth-First Search (BFS)](breadth-first-search.md)**: Level-by-level traversal.
*   **[Depth-First Search (DFS)](depth-first-search.md)**: Deep branch-by-branch traversal.
*   **[Dijkstra's Algorithm](dijkstras-algorithm.md)**: Single-source shortest path finder.

### Sorting Algorithms
*   **[Selection Sort](selection-sort.md)**: Simple sorting via minimum selection.
*   **[Quicksort](quicksort.md)**: Fast sorting using pivot partitioning.
*   **[Merge Sort](merge-sort.md)**: Stable sorting via divide-and-conquer.

## 4. Algorithmic Paradigms & Techniques
*   **[Recursion](recursion.md)**: Solving problems by calling the same function.
*   **[Brute Force](brute-force.md)**: Simple trial of all possible candidates.
*   **[Divide and Conquer](divide-and-conquer.md)**: Breaking problems into independent subproblems.
*   **[Greedy Algorithm](greedy-algorithm.md)**: Making locally optimal choices at each step.
*   **[Backtracking](backtracking.md)**: Incremental search that backtracks on failure.
*   **[Dynamic Programming](dynamic-programming.md)**: Storing solutions to overlapping subproblems.
*   **[Two Pointers](two-pointers.md)**: Traversing arrays using two cursor indices.
*   **[Sliding Window](sliding-window.md)**: Processing sub-segments using a moving window.
*   **[Approximation Algorithm](approximation-algorithm.md)**: Near-optimal solutions to hard optimization problems.
*   **[K-Nearest Neighbors (KNN)](k-nearest-neighbors.md)**: Distance-based classification algorithm.
